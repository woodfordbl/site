# Personal Site

Hi, I'm [Blake](https://github.com/woodfordbl). This is my corner of the internet — and, honestly, my excuse to play with the [TanStack](https://tanstack.com/) ecosystem.

**Live:** [site-five-bice-36.vercel.app](https://site-five-bice-36.vercel.app)

It started as a place to point people to. It's turning into something more fun: a **local-first, Notion-ish workspace** — a block canvas with pages, embedded databases, and charts, where everything you touch saves locally first and stays instant, even offline.

The part I'm actually excited about is under the hood. It runs on [TanStack DB](https://tanstack.com/db), which leaves a clean seam for a real sync engine. The plan: drop in [ElectricSQL](https://electric-sql.com/) + Postgres and turn this into a real-time, multi-device, endlessly malleable notetaking and data-viz playground — without giving up the instant local feel.

For now it's small, personal, and a work in progress. That's kind of the point.
